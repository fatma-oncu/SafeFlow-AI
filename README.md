# SafeFlow AI

AI-powered Occupational Health & Safety Operations Platform

## Overview

SafeFlow AI is a modular Occupational Health & Safety platform designed for enterprises and OSGBs.

The platform provides:

- Hybrid Training Management
- Employee Management
- Certificate Management
- Dashboard
- Multi-Tenant Architecture
- Offline Mobile Inspections
- AI-assisted Decision Support

## Technology Stack

- ASP.NET Core 8
- Entity Framework Core
- SQL Server
- Flutter
- Python
- Docker

## Architecture

- Clean Architecture
- Domain Driven Design
- CQRS
- MediatR
